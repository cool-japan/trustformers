# TrustformeRS Framework Demos

This directory contains comprehensive examples demonstrating TrustformeRS integration with popular JavaScript frameworks and platforms.

## Demo Applications

### 1. React Application Examples
- **react-sentiment-analyzer**: Complete React app with sentiment analysis
- **react-chatbot**: Real-time chatbot with TrustformeRS
- **react-text-classifier**: Text classification with live updates

### 2. Vue.js Integration Demos  
- **vue-document-analyzer**: Vue 3 composition API demo with document analysis
- **vue-translation-app**: Multi-language translation interface
- **vue-model-playground**: Interactive model testing playground

### 3. Node.js Server Examples
- **node-api-server**: REST API server with TrustformeRS endpoints
- **node-websocket-chat**: Real-time chat server with AI processing
- **node-batch-processor**: Batch text processing service

### 4. Electron Desktop App
- **electron-ai-assistant**: Cross-platform desktop AI assistant
- **electron-text-editor**: Smart text editor with AI suggestions

### 5. Mobile Web App
- **mobile-pwa**: Progressive Web App optimized for mobile devices
- **mobile-voice-assistant**: Voice-enabled mobile AI assistant

## Running the Demos

Each demo includes its own README with specific setup instructions. Most demos require:

1. Node.js 16+ 
2. npm or yarn
3. Modern browser with WebAssembly support

## Prerequisites

```bash
# Install TrustformeRS JS package
npm install @trustformers/js

# Or use from local build
npm run build
```

## Quick Start

```bash
# Navigate to any demo directory
cd framework-demos/react-sentiment-analyzer

# Install dependencies
npm install

# Start development server  
npm run dev
```
